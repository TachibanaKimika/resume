# Resume
> 简单快速的markdown简历制作工具, 访问 [https://tachibanakimika.github.io/resume/](https://tachibanakimika.github.io/resume/)
## Features
- [x] local storage本地保存
- [x] 支持自定义CSS, 同时管理多个CSS
- [x] 支持多版本简历共存
- [x] 实时浏览
- [x] PDF导出
